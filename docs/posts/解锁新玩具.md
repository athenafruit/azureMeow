# 解锁新玩具

小红书上有挺多猫脸人身的AI视频，每次看这种视频就想笑。于是看多了就想西卡卡和西兰花猫脸人身的样子。(西卡卡和西兰花是我的猫）so，心动不如行动。于是和DeepSeek聊了聊该怎么做，流程大致如下：![](/images/wechat/解锁新玩具_d4d259f78555.jpg)下午用了三个多小时搞完了前三步，核心主要内容是训练了一个lora模型，然后测试了下生成效果。mark下教程，对于一个此前没有搞过算法训练的小白来说，我还挺满意的hhh第一步：准备数据集。其实现在国内的训练平台非常全面，数据集准备图片就好。格式最好是JPG/JPEG/PNG，数量10～50张为好。鉴于薅算力，训练平台有针对新用户的免费训练次数或算力，所以我用了10张。原本准备了30+张图片，但是消耗算力超出了免费额度，贫穷让我放弃。此外这里也遇见了一个小坑：iCloud下载的图片默认不是JPEG格式，需要在下载时额外设置。第二步：在合适的训练平台上训练，此处基座模型、裁剪、自动打标，训练可以一条龙完成。我在此过程中使用了两款工具TensorArt和LibLibAI，贴下对比见下图。TensorArt需要梯子，每天登录有算力，省着点用一天可以训一次；LibLibAI每月五次，不需要梯子。起初先使用的LibLibAI，但是在洗数据打标过程中总是失败，中间花了点时间阅读原理和文档，然后不撞南墙的去使用了另一个平台。整体使用流程没有明显太大差异，原理相似。特性维度| **TensorArt（综合商场）**| **LibLibAI（定制工坊）**| **意味着什么**  
---|---|---|---  
**核心定位**| **一站式AIGC平台**| **AI模型分享与训练社区**|  训练模型到生图生视频都需要，但TensorArt可能更“省心”  
**模型训练**|  支持，功能直观，**与生成流程无缝集成**| **极其专业和深入** ，提供**超多高级参数** 可调| LibLibAI可能训练出更精细的模型，但需要更多知识  
**模型社区**|  有，但并非绝对核心| **绝对的核心和优势** ，海量用户分享的优质模型| 在LibLibAI更容易找到别人的“猫猫Lora”参考  
**生成功能**| **非常强大** ，支持文生图、图生图、**视频生成** 、局部重绘等| 支持文生图、图生图，但**更侧重于“测试模型”**| **TensorArt胜出** ，因为最终需要视频生成功能  
**用户体验**|  界面现代，功能整合度高，**对新手友好**|  界面更“极客”，功能围绕模型展开，**对进阶用户更友好**| **TensorArt上手更快** ，学习成本更低  
**收费方式**|  提供免费额度，付费购买“积分”进行计算| 类似，也提供免费额度，付费购买“电量”| 两者性价比相近，均可接受  
第三步，处理好数据就可以自动训练啦。训练完大概20min。我训练了三次：\- 第一次因为数据集格式问题失败了，我还跟好友吐槽了这个问题不能前置告知，浪费了一次训练次数。她吐槽此前遇见过我这类用户哈哈哈！果然要想做朋友是不能合作的🤣\- 第二次忘了修改默认触发词，结果我拿猫的数据集训练了一个女孩子，伐开心！\- 第三次总算成功啦。见下图![](/images/wechat/解锁新玩具_4034287d25f8.jpg)03和05和西卡卡简直一模一样，开心！拿模型测试了几张效果：西卡卡背着书包![](/images/wechat/解锁新玩具_6dbd1bdce0bd.jpg)西卡卡看书![](/images/wechat/解锁新玩具_57ec23134cba.jpg)西卡卡喝咖啡![](/images/wechat/解锁新玩具_dd555d158a19.jpg)
