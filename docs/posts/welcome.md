# 欢迎来到我的博客

> 这是博客的第一篇文章 🎉

## 为什么写博客

写博客是一种很好的学习方式：

1. **输出倒逼输入** - 写作让你更深入地思考
2. **建立知识体系** - 将零散的知识串联起来
3. **分享与交流** - 与更多人交流想法

## 博客技术栈

这个博客使用以下技术搭建：

- **VitePress** - 基于 Vue 的静态网站生成器
- **GitHub Pages** - 免费的静态网站托管
- **GitHub Actions** - 自动化部署
